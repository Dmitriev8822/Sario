# Sario

Sario — лёгкая браузерная пиксельная мини-игра-подарок, вдохновлённая классическими платформерами.
Проект написан на чистых HTML, CSS и JavaScript, поэтому его можно открыть локально или разместить на обычном статическом хостинге.

## Запуск

Откройте `index.html` в браузере или запустите любой локальный статический сервер:

```bash
python3 -m http.server 8080
```

После этого перейдите на `http://localhost:8080`.

## Структура

- `index.html` — основной HTML-каркас, главный экран, экран игры и информационные секции.
- `src/styles/main.css` — общий пиксельный стиль, адаптивность и подключение фоновой картинки.
- `src/scripts/main.js` — навигация между экранами и запуск игры.
- `src/scripts/game.js` — логика платформера: движение, прыжки, сбор звёзд, победа.
- `src/assets/images/pixel-party-bg.svg` — заменяемый фон проекта.

## Настройка под именинника

- Замените текст поздравления в `src/scripts/game.js` в методе `checkGift`.
- Чтобы поменять настроение страницы, замените файл `src/assets/images/pixel-party-bg.svg` или укажите другой путь в CSS-переменной `--bg-image`.
- Для встраивания в готовый сайт перенесите секцию `#game`, `canvas`, стили и два JavaScript-файла.
