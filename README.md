# Sario

Sario — лёгкая браузерная пиксельная мини-игра-подарок, вдохновлённая классическими платформерами.
Проект написан на чистых HTML, CSS и JavaScript, поэтому его можно открыть локально или разместить на обычном статическом хостинге.

## Запуск

Откройте `index.html` в браузере или запустите любой локальный статический сервер:

```bash
python3 -m http.server 8080
```

После этого перейдите на `http://localhost:8080`.

## Структура

- `index.html` — основной HTML-каркас, главный экран, экран игры, topbar и панель редактора уровня.
- `style.css` — общий пиксельный стиль, адаптивность, canvas, topbar и элементы редактора.
- `game.js` — конфигурация уровня, движение персонажа, коллизии, редактор блоков/предметов, сохранение уровня и ачивки.
- `assets/player/young/` — PNG-спрайты персонажа.
- `assets/attributes/` — PNG-картинки предметов, которые можно разместить на уровне и показать при сборе.

## Настройка уровня

Все стартовые данные уровня лежат в `CONFIG.level` внутри `game.js`:

```js
level: {
  blocks: [
    { x: 360, y: 286, w: 124, h: 18 },
  ],
  items: [
    {
      x: 760,
      y: 202,
      w: 34,
      h: 34,
      src: "assets/attributes/example.png",
      title: "Новая ачивка",
      text: "Ты нашёл предмет уровня!",
    },
  ],
}
```

- `blocks` — платформы уровня. `x` и `y` задают позицию, `w` и `h` — размер.
- `items` — собираемые предметы. `src` должен указывать на PNG из `assets/attributes/`, а `title` и `text` показываются в карточке ачивки при сборе.

## Редактор уровня в игре

1. Нажмите `Начать игру`.
2. Нажмите кнопку `Редактор` в верхней панели.
3. Выберите инструмент `Блок` или `Предмет`.
4. Кликните по canvas, чтобы добавить объект.
5. Используйте `Alt + клик`, `Shift + клик` или правую кнопку мыши, чтобы удалить объект выбранного типа.
6. Нажмите `Экспорт`, чтобы скопировать JSON уровня и при необходимости перенести его в `CONFIG.level`.
7. Нажмите `Сброс`, чтобы удалить сохранённый в браузере вариант и вернуться к дефолтному уровню.

Изменения редактора автоматически сохраняются в `localStorage` браузера под ключом `sario.level`.
